Decoration

Just as a nexus can have many tags, a tag can have many decorations. A decoration qualifies how the tag applies to the nexus. For example, in the nexus 'Battle of Waterloo', a 'person' tag for 'Napoleon' is attached. This tells us that Napoleon was associated with the event, but how? The decorations tell us: there is a 'role' decoration with the value 'commander', and a 'world actor' decoration with the value 'France'. Now we know that Napoleon's role at the Battle of Waterloo was as commander of the French.

There are different rules for which decorations are allowed on which tags. For the most part, the user interface tries to make this so easy you don't even need to think about it.

All tags, except Meta tags, can have multiple 'see_also' decorations, these can point to any other tag or nexus, or even be a free-form URL or text reference.

The 'event type' and 'idea' tags do not take any other decorations. The decorations you can put on the other kinds of tags are described in the table below.

tag typedecorations
event
relationship how this event relates to the one named in the tag. For example, the Battle of Gettysburg might have an 'event' tag named 'American Civil War'. That tag would have a relationship decoration whose value was 'part of', indicating that this nexus was a part of some larger event.
world actor
role the role that this world actor played in relation to the nexus
person
role the role that this person played in relation to the nexus
world actor the world actor that this person was associated with for the nexus
result
world actor the world actor that this result applies so, usually names one of the world actor tags on the nexus
impact
world actor the world actor that this impact applies so, usually names one of the world actor tags on the nexus
estimate a number that quantifies this estimate of this impact
high estimate optionally, a number that quantifies the higher-bound of estimates for this impact
low estimate optionally, a number that quantifies the lower-bound of estimates for this impact
citation
excerpt a brief snippet from the citation, as it pertains to a fact described in the nexus
meta
value a value that is only meaningful for things that care about meta tags
