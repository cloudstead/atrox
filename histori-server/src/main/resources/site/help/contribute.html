Contribute to histori.city

In the near future, there will some very important ways that everyone can contribute:

  • Adding new historical information to the map

    histori.city is intended to be a crowd-sourced effort. Your participation will be vital in filling the map and making it rich with history.

  • Correcting historical information on the map

    Our current data has been auto-imported from Wikipedia. Our import scripts are not perfect. Sometimes they make mistakes. By cleaning up some of the existing map data, you help to make the service more useful to everyone.

For the moment, the most important things we need help with are:

  • Spreading the word

    Do you like histori.city? Do you know someone who would? Is there a history teacher in your life that should know about this? Any amateur historians among your friends? Professional historians? Let other people know about histori.city! Send someone a permalink to a cool search you've crafted. They might get a kick out of it too.

  • Front-end development

    I'm not much of a JavaScript/HTML/CSS developer. I've built histori.city as simply and cleanly as I could. I know I've probably done some foolish things that an expert coder will laugh at, but I'm pretty sure that same developer would also find the code to be clean, straightforward and relatively easy to follow.

    I've tried to make the codebase attractive to a seasoned developer who wants to make it their own and implement an Ember/Angular/React framework on top of (or in replacement of) the lightweight jQuery code in place now. If you're interested, please let me know.

  • More Wikipedia importers

    We have importers for battles and city/settlement foundings, but there could be so many more. Writing an importer is not terribly hard; making the importer more useful by extracting more tags and decorations from Wikipedia articles can be a lot of fun.

    I've pulled together some basic instructions on working with Wikipedia archives, which includes information about writing importers. Give it a whirl! If you get stuck we can hash things out on Developers section of our discussion forums.

  • Financial support

    It costs money to operate the histori.city service. Your contributions are a huge help in supporting our ongoing operations.

    Please consider making a donation today. We use PayPal so it's very fast and easy. A donation of any amount is sincerely appreciated. We are exploring ways to become a non-profit, but for now these donations are not tax-deductible.
